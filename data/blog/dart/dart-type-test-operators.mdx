# **is** / **is !**

```dart
void main() {
  String value1 = "Dart Tutorial";
  int age = 10;

  print(value1 is String);
  print(age is !int);
}

// true
// false
```

# **runtimeType**

```dart
void main() {
  int number = 20;

  print("Type of number is ${number.runtimeType}");
}

// Type of number is int
```
