# isEmpty / isNotEmpty

```dart
void main() {
   String str = "Hi";
   print(str.isEmpty);     //Example of isEmpty
   print(str.isNotEmpty);  //Example of isNotEmpty
}
```

> isEmpty and isNotEmpty are properties of string, not methods
